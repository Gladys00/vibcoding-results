## Version Habit

- `main` 保存已经能运行的稳定版本。
- 每个工具放在独立目录中，互不影响。
- 每次工具迭代先更新对应目录下的 `CHANGELOG.md`，记录版本、日期和主要变化。
- 后续开发可以使用类似 `ledgerforge/v1.3.3` 的分支，确认后再合并回 `main`。
