# 版本记录

本文档记录 docs 维护视角下的版本变化。根目录 `CHANGELOG.md` 仍是代码侧已有版本记录来源。

## V1.3.2.2

状态：当前本地代码版本。

类型：功能完善与问题修复版本。

已确认变更：

- Excel 导入向导去掉“归属”步骤，导入流程不再要求用户在导入时做项目归集。
- 导入预览页改为展示本次真实解析出的 Sheet、项目、台账明细、收付款、发票和问题。
- 导入修正页改为展示真实解析问题，并支持为每条问题选择处理方式。
- 导入确认页改为基于真实解析数据展示确认摘要。
- 没有真实解析文件时，预览、修正、确认页不再展示 mock 示例。
- 新增 `EXCEL_IMPORT_PREVIEW_ISSUES.md` 和 `EXCEL_IMPORT_CORRECTION_LOGIC.md`。

影响范围：

- Excel 导入向导。
- Excel 导入预览。
- Excel 导入修正。
- Excel 导入确认。
- 导入相关维护文档。

已确认未改变：

- 未新增后端。
- 未新增数据库。
- 未新增 API。
- 未改变项目台账、统计归集、报表查询的数据存储方式。

未知：

- 当前版本是否已经推送到远程仓库。
- 当前版本是否已经创建远程标签。

## V1.3.2.1

状态：历史本地代码版本。

类型：修复版本。

已确认变更：

- 修复 Excel 导入历史页中台账金额表头和帮助提示被遮挡的问题。
- 完善 Excel 导入 Sheet 选择：
  - 支持全选。
  - 支持多选。
  - 支持取消单个 Sheet 选中。
  - 支持清空选择。
  - 支持离开页面后返回仍保留选择状态。
- 调整 Excel 导入映射页操作：
  - “调整”不再打开新增字段弹窗。
  - “调整”改为选择系统已有字段进行映射。
  - 不允许在该入口新增自定义字段。

已确认未改变：

- 未新增后端。
- 未新增数据库。
- 未新增 API。
- 未改变整体页面布局。
- 未改变主导航结构。

未知：

- 当前版本是否已经推送到远程仓库。
- 当前版本是否已经创建远程标签。
